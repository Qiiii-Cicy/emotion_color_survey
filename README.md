# 情绪与颜色对应关系调查系统

这是一个用于收集用户对情绪与颜色对应关系看法的前后端调查系统。

## 功能特点

- **六种情绪**：愤怒(anger)、快乐(enjoyment)、惊讶(surprise)、恐惧(fear)、厌恶(disgust)、悲伤(sadness)
- **直观的颜色选择器**：用户可以通过HTML5颜色选择器为每种情绪选择对应的颜色
- **进度跟踪**：实时显示调查完成进度
- **数据收集**：所有提交的数据自动保存到JSON文件中

## 文件结构

```
emotion-color/
├── index.html      # 前端主页面
├── style.css       # 样式文件
├── script.js       # JavaScript交互逻辑
├── submit.php      # PHP后端处理文件
├── data.json       # 数据存储文件（自动生成）
└── README.md       # 说明文档
```

## 使用方法

### 1. 环境要求

- Web服务器（Apache、Nginx等）
- PHP 7.0 或更高版本

### 2. 部署步骤

1. 将所有文件上传到Web服务器目录
2. 确保Web服务器有写入权限（用于创建和更新`data.json`文件）
3. 在浏览器中访问`index.html`

### 3. 使用流程

1. 用户打开网页后，会看到六个情绪卡片
2. 为每种情绪选择对应的颜色
3. 点击"确定"按钮确认选择
4. 完成所有情绪的颜色选择后，点击"提交调查结果"
5. 数据会自动保存到`data.json`文件中

## 数据格式

`data.json`文件存储所有用户的提交数据，格式如下：

```json
[
  {
    "anger": "#ff0000",
    "enjoyment": "#ffff00",
    "surprise": "#ff00ff",
    "fear": "#000000",
    "disgust": "#00ff00",
    "sadness": "#0000ff",
    "timestamp": "2024-01-01 12:00:00"
  }
]
```

## 数据分析

收集到足够数据后，您可以：

1. 读取`data.json`文件
2. 对每种情绪的所有颜色值进行统计分析
3. 计算平均值或最常见的颜色值
4. 确定每种情绪对应的最终颜色

## 注意事项

- 确保`data.json`文件所在目录有写入权限
- 建议定期备份`data.json`文件
- 如果数据量很大，可以考虑使用数据库替代JSON文件

## 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **后端**：PHP
- **数据存储**：JSON格式
